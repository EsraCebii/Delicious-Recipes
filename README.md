### Rest Api

[api linki ](https://esra-recipe.herokuapp.com/)

### Api Endpoints 

```
GET    /recipes
GET    /recipes/:id/comments
GET    /comments
GET    /favs
GET    /recipes/:id
POST   /recipes
POST   /favs
POST   /comments
DELETE /recipes/:id
DELETE /favs/:id
DELETE /comments/:id
```
